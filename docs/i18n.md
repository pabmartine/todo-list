# Internationalization

The project uses gettext.

## Files

- Source code: `src/todo_list/`
- Template catalog: `locale/todo-list.pot`
- Translations: `locale/<lang>/LC_MESSAGES/todo-list.po`
- Compiled catalogs: `locale/<lang>/LC_MESSAGES/todo-list.mo`

## Commands

Refresh catalogs from source:

```bash
bash extract_strings.sh
```

Compile `.po` files into `.mo`:

```bash
bash compile_translations.sh
```

## Runtime Behavior

Locale bootstrap is handled in `src/todo_list/i18n.py`.

The app supports:

- Auto-detect
- English
- Spanish

Language can be changed at runtime from the application menu/preferences.
