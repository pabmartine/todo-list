# User Guide

## Main Concepts

- A task can have title, notes, project, date, completion state, favorite flag and subtasks.
- A project groups tasks and has a color.
- Smart lists are generated views, not separate task stores.

## Smart Lists

- `Today`: tasks dated today
- `Next 7 days`: tasks scheduled within the next week
- `All`: all incomplete tasks
- `Overdue`: incomplete tasks with a past date
- `Favorites`: starred incomplete tasks
- `Archived`: completed tasks

## Basic Usage

### Create A Task

Type in the `New task...` entry and press Enter.

### Edit A Task

Select a task to open its detail view. Depending on the window width, it appears as a side panel or a small-screen detail page.

You can edit:

- title
- date
- notes
- project
- completed state
- subtasks

### Favorite A Task

Use the star button in the task row.

### Manage Subtasks

Open a task and use the `Subtasks` section to:

- add a new checklist item
- mark a subtask as completed
- rename a subtask
- delete a subtask

Tasks with subtasks show a small completion summary in the main list.

### Search Tasks

Use the `Search tasks...` field above the list to filter tasks within the current view.

### Manage Projects

Use the add/edit/delete project actions from the sidebar header and project views.

### Reorder Tasks

Drag a task by its handle and drop it over another task in the same visible list.

## Preferences

From the application menu you can change:

- interface language
- dark theme

Window size and selected list are remembered automatically.

## Import And Export

From the application menu you can:

- import tasks from a JSON backup
- export the current data to a JSON file
